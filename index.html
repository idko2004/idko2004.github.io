Secret Santa

Secret santa is a Discord bot for doing a... secret santa.

The way it works is:

  • It sends a message in the specified channel.
  • The people who wants to participate react to the message with a specific emoji.
  • If the participant has the requiered role (optional), another role is given to them.
  • You should set up that role in a way that it gives access to a secret santa channel.
  • When you click the send santas button, the participants will receive their santa in their DMs (please tell everyone to keep their DMs open).

convertMusic

convertMusic is a little node.js program that converts files in a folder to another format using ffmpeg.

Like, for example, imagine that your car has a cheap stereo system that only works with mp3 files, but all your music files are in opus, instead of converting them one by one with ffmpeg, you can just use this program.

This program is basically a glorified way of running "ffmpeg -i input.opus output.mp3", I mean, that's literally what it does, but for each file in a folder.
